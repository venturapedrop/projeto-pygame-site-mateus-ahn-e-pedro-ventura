How to Play

Main Menu

S - Start game

Q - Quit

L - Switch language

During the Game

Type letters or words to stop enemies.

Pause Menu

ESC - Resume

M - Main menu

Score Screens

R - Restart

Enter - Confirm

Technical Features

Features

  • Object-oriented design
  • Sprite collision
  • Screen-based states
  • Extensive docstrings

Requirements

  • Python 3.7 or newer
  • Pygame library

Installation

  1. Clone or download
  2. Install Pygame
  3. Run python3 game.py
